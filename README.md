乌云凉沉默着，没有说话。

“楚阳，他才三个月大，我就将他捡了回来，带在自己身边，一点一点养大，这么多年来，与其说我是他的师傅，倒不如说我就是他的父亲，他的母亲！”

“他的名字，是我取的；楚阳，初阳，出阳，代表了我一生最大的希望！大师兄，你明白么？”

“我明白。”乌云凉沉沉地叹了口气。

“你不明白！”孟超然萧索的站着，突然转身，毫无预兆的，重重的一拳砸在乌云凉的脸上。噗的一声，乌云凉身形后仰，却不出一声。
